# 《美术奕论》简介

视频《美术奕论》属于广州美术学院创新强校项目 “精品资源共享课《视觉艺术原理》”，项目负责人为广州美术学院朱松青教授。该视频把视觉艺术原理（造型学）的知识点，以通俗简短的形式进行普及，旨在推动美术教育的建设与改革，建立一套立足美术原理体系的美术教育体系。奕，本义是大，后引申为非一般的，又有美好、光明、积累等意思。《美术奕论》，就是要为我们美术教育的美好明天标新立异、建言献策、添砖加瓦。

# 01 盲人摸象

在美术界，人们对美术的认识基本上就是盲人摸象，因为我们迄今的美术教育都是碎片式的教育。

# 02 男生女生

男女孩接触物质材料的频率差异，就导致男女生学美术的几率差异。

# 03 普通话

既通行于高精尖又活用于大众化的 “普通话”—— 造型学。

# 04 万能的支点

掌握了造型学这个万能的支点，不管你是专业人士还是普通百姓，都可以用来撬动美术世界！

# 05 茶壶

学和术，就有点像茶叶和水，两者结合起来，才有茶喝。造型学则是把美术的学与术结合起来的一个兼容器。

# 06 以讹传讹

美术原理和黄金比一样，初中生一听就懂。

# 07 画饼充饥

真是科学的终极目标；美才是艺术的普遍追求。用真的标准苛求艺术苛求美，本身就不科学。

# 08 差不多

目前很多美术课本都说三原色是红黄蓝，它明显违反了科学原理。

# 09 生活

有了美术生活和生活美术这两个概念，我们的认识和表达就简明了。
